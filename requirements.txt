• 📋 Installation Instructions

### Basic Installation
```bash
pip install -r requirements.txt
```

• Development Installation
```bash
pip install -r requirements.txt
pip install pytest black flake8 mypy
```

• Production Installation
```bash
pip install -r requirements.txt --only=main
```
> **Note:** Consider using separate `requirements-dev.txt` for development dependencies
